获取方式：
先遍历获取mdx内容中所有符合条件的标题（h1，title）与tocs（h2-h5，tocs），然后把这些信息都放入到mdx的元信息中
```javascript
tree.children.push({
  type: 'mdxjsEsm',
  value: inserted,
  data: {
    estree: parse(inserted, {
      ecmaVersion: 2020,
      sourceType: 'module'
    })
  }
} as MdxjsEsm);
```
我们就可以在后面通过导入的方式获取到这些信息
